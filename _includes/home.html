The National Information Exchange Model (NIEM) is a community-driven, common vocabulary that enables efficient information exchange across diverse public and private organizations. NIEM can save time and money by providing consistent, reusable data terms and definitions and repeatable processes. This GitHub site is for the community to have access to resources that will help them use NIEM and make NIEM easier for others to use.
